- Receive push notifications when one of your favorite podcast shows does a live show

Changelog
- The color gradient in the podcast detail view has been improved. It is not that dark any more.
- The live player design was completly redone from scratch
- I massively improved the VoiceOver support
- Bugfix: The event date view now keeps its state when the user scrolls to a longer list
- The event date view got updated to not only show the days left until the event takes place but also hourse and minutes if the event gets closer
- The event tab got renamed to Calendar as it fits better
- User who use the app a lot will see a popup one single time which tells them how to support the development of the app
- A button has been added to settings which makes it very easy to rate the app in the Appstore.
